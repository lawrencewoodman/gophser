Gophers
=======

A [Gopher](https://en.wikipedia.org/wiki/Gopher_(protocol)) server written in Tcl

Requirements
------------
*  Tcl 8.6+


Testing
-------
There is a testsuite in `tests/`.  To run it:

    $ tclsh tests/*.test.tcl
